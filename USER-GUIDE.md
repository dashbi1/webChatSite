# 工大圈子 - 使用指南

## 启动服务

```bash
# 1. 启动后端（在 server 目录）
cd server
npm start
# 后端运行在 http://localhost:3000

# 2. 启动前端（在 client 目录，新开终端）
cd client
npm run dev:h5
# 前端运行在 http://localhost:5173
```

## 普通用户

### 访问地址
浏览器打开 **http://localhost:5173**

### 注册
1. 首页自动跳转到登录页
2. 点击"立即注册"
3. 输入手机号（11 位，13x/15x/17x/18x 开头）
4. 点击"获取验证码"，输入 **123456**（测试阶段固定验证码）
5. 设置密码，点击注册

### 登录
- 手机号 + 密码登录

### 核心功能
| 功能 | 操作 |
|------|------|
| 浏览帖子 | 首页自动加载，下拉刷新，上拉加载更多 |
| 发帖 | 首页右下角 **+** 按钮，支持文字+图片（最多9张） |
| 编辑/删除帖子 | 自己的帖子右上角 **···** → 编辑/删除 |
| 搜索用户 | 首页右上角 🔍，输入昵称搜索 |
| 添加好友 | 搜索用户 → 点击头像进入资料页 → 添加好友 |
| 处理好友申请 | 好友页 → 好友申请 → 接受/拒绝 |
| 点赞/评论 | 好友的帖子可点赞和评论，非好友不可 |
| 私聊 | 好友列表 → 点击好友 → 进入聊天（实时消息） |
| 通知 | 任意页面右上角 🔔 铃铛，红点表示有新通知 |
| 编辑资料 | "我的" → 编辑资料 → 修改头像/昵称/学院/年级 |
| 举报 | 他人帖子 **···** → 举报；用户资料页 → 举报用户 |
