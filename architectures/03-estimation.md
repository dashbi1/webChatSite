# 工大圈子 - 工作量估算

> Lab2 要求：使用 Story Point 估算工作量，单位小时
> 预定值：1/2、1、2、3、5、8、13、20
> 团队成员分别估计，差异大时讨论达成共识

## 估算说明

- 3 名团队成员分别独立估算，经讨论后达成共识
- 工作量包含：前端 UI + 后端 API + 数据库 + 测试
- 由于使用 uni-app 一套代码多端，不额外计算各平台适配工作量

## 估算依据

- **US09 (私聊) 最高 13h**：涉及 WebSocket 服务搭建、消息存储、实时推送、聊天 UI，技术复杂度最高
- **US01 (信息流) 8h**：分页加载、下拉刷新、帖子卡片组件、好友/非好友状态判断
