# 工大圈子 - 项目需求分析与规划总览

> 本文件是需求分析阶段的总纲，汇总所有决策和待完成任务。
> 后续每个子文件对应 Lab2 实验报告的一个章节。

## 项目基本信息

- **项目名称**：工大圈子
- **项目定位**：面向哈工大学生的校园社交平台，用户可发布动态、添加好友、私聊互动
- **目标用户**：哈工大在校学生（实际不限制，手机号即可注册）
- **团队**：3 人（代码由 1 人完成，2 人名义参与）
- **测试规模**：100 人以内

## 技术栈决策

| 层 | 技术 | 说明 |
|----|------|------|
| 前端 | uni-app (Vue 3) | 一套代码 → Web H5 + Android App + 微信小程序 |
| 后端 | Node.js (Express/Koa) | REST API + Socket.io 实时通信 |
| 数据库 | Supabase (PostgreSQL) | 免费云端，含 Auth + Storage + Realtime |
| 文件存储 | Supabase Storage | 头像、图片、视频 |
| 部署 | 2C4G VPS | Nginx 反代 + Node 服务 + uni-app 静态资源 |
